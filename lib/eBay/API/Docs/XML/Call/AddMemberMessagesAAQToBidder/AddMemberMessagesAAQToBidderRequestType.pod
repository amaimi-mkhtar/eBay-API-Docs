=head1 NAME

eBay::API::XML::Call::AddMemberMessagesAAQToBidder::AddMemberMessagesAAQToBidderRequestType

=head1 DESCRIPTION

Enables a seller to send up to 10 messages to bidders and
users who have made offers (via Best Offer) during an
active listing. The call can be used with Motors or Business & Industrial 
categories only.
