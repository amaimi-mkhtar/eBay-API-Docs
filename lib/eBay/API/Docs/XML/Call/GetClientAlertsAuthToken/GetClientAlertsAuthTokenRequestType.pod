=head1 eBay::API::XML::Call::GetClientAlertsAuthToken::GetClientAlertsAuthTokenRequestType

eBay::API::XML::Call::GetClientAlertsAuthToken::GetClientAlertsAuthTokenRequestType

=head1 DESCRIPTION

Retrieves a token required for the GetUserAlerts call in the Client Alerts API. Required input is a
valid user token, which is the same user token used for authentication and authorization in the
Trading API. Client Alerts token also required for Client Alerts API calls Login and Logout.
